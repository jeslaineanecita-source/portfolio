# Professional Portfolio

A basic web-based professional portfolio showcasing skills, projects, experience, and contact information.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Smooth Navigation**: Scroll to sections with smooth animations
- **Interactive Elements**: Hover effects, form validation, and scroll animations
- **Modern Styling**: Gradient headers, card-based layout, and clean typography

## Sections

1. **About Me**: Personal introduction with profile picture
2. **Skills**: Grid of technical and professional skills
3. **Projects**: Showcase of notable projects with images and tags
4. **Experience**: Timeline of work history and career progression
5. **Contact**: Form for sending messages and social media links

## Technologies Used

- HTML5 for structure
- CSS3 for styling (including animations and responsive design)
- JavaScript for interactivity
- Font Awesome for icons
- Google Fonts (via CDN)

## Setup

1. Clone this repository
2. Open `index.html` in a web browser
3. Navigate through the sections using the navigation bar
