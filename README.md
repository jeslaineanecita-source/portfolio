# Jeslaine Anecita Portfolio

A professional portfolio website showcasing the work and skills of Jeslaine Anecita, a third-year ICT student at Iloilo State University of Fisheries Science and Technology.

## Overview

This portfolio demonstrates modern web development skills, featuring a clean, professional design with smooth animations and responsive layout. It serves as both a personal brand showcase and a practical demonstration of technical capabilities.

## Key Features

- **Responsive Design**: Fully responsive layout for desktop, tablet, and mobile devices
- **Dark Theme**: Professional dark theme with gradient accents
- **Smooth Animations**: Multiple animation effects for an engaging user experience
- **Interactive Elements**: Hover effects, form validation, and dynamic UI elements
- **Modern Technology Stack**: Built with HTML5, CSS3, and JavaScript
- **Professional Presentation**: Suitable for job applications and client presentations

## Sections

1. **Hero Section**: Introduction with name, title, and call-to-action buttons
2. **About Me**: Personal background, education, and career goals
3. **Technical Skills**: Comprehensive skill assessment with progress bars
4. **Featured Projects**: Showcase of notable academic and personal projects
5. **Academic Journey**: Timeline of educational experience at ISUFST
6. **Contact**: Contact form and social media links

## Technologies Used

- HTML5 for semantic structure
- CSS3 with custom properties for theming
- Vanilla JavaScript for interactivity
- Font Awesome for icons
- Google Fonts for typography
- Live Server for local development

## Development Setup

### Prerequisites

- Node.js (version 14 or higher)
- Live server for local development

### Installation

```bash
cd portfolio
npm install
```

### Development

```bash
npm run dev
```

This will start the development server on http://localhost:8080

### Build

```bash
npm run build
```

This will prepare the portfolio for deployment

## Project Structure

- `index.html` - Main portfolio page
- `styles.css` - CSS styling with dark theme and animations
- `script.js` - JavaScript for interactivity and animations
- `package.json` - Project configuration and scripts
